# 七處徵心 · Where Is the Mind?

《大佛頂首楞嚴經》卷一「七處徵心」的**中英雙語課堂互動遊戲**，供成人佛學班理解這段論辯：
阿難七次指出「心」的所在，世尊七次破之，最終顯出——心了不可得，破妄方能顯真。

— © 寶嚴禪寺 · 星慕法師　課堂教學用

## 兩種玩法
- **講解模式**：老師帶讀，先讓全班討論「世尊會怎麼破？」，再揭曉破斥。
- **闖關模式**：分組搶答，三選一選出佛陀的破斥，即時計分（7/7）。

## 內容
- 七處逐一呈現：阿難主張＋經文＋世尊破斥＋關鍵句，配身像 SVG 逐處打✕。
- 結局開示「妄心 vs 真心、認賊為子」，附三題課堂省思。
- 中英雙語切換；投影鍵盤操作（1/2/3 選答、Enter／→ 下一步）。
